# 💪 FitBuddy - Full Stack Gym Management Website

A modern full-stack gym management platform that allows users to register, log in, explore membership plans, enroll in courses, make secure online payments, and print payment receipts.

- User Registration & Login
- Secure Password Encryption (bcrypt)

- Course Enrollment
- Razorpay Payment Gateway Integration
- Payment Verification
- Printable Payment Receipt
- Contact Form
- MongoDB Atlas Cloud Database

- REST API Architecture

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/FATE24-ChL/FitBuddy_FULL-STACK-GYM-WEBSITE.git
```

Install frontend dependencies

```bash
cd client
npm install
npm run dev
```

Install backend dependencies

```bash
cd server
npm install
npm start
```
